# LowEnd Forge

A bass-first transcription workbench for turning low-end audio into editable tab.

Product page: https://hannes-software.com/lowend-forge/
Downloads: https://hannes-software.com/lowend-forge/#downloads
Latest release: https://github.com/hannes-wan/lowend-forge-official/releases/tag/v0.1.3
